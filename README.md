# 📬 TG Bot Schedule — Telegram-бот для расписания колледжа

## Проблема

В колледже расписание занятий приходит на почту от учебного отдела

## Решение

Бот автоматически решает эти проблемы:

1. **Каждые 5 минут** проверяет почту на новые письма от учебного отдела
2. **Парсит** полученное расписание и **переводит "пары" в "уроки"** с точным временем начала и конца
3. **Дополняет** расписание актуальным расписанием автобусов:
   - на какой автобус успеть, чтобы приехать к нужному уроку
   - какие автобусы идут после последнего урока
4. **Отправляет** готовое, красиво отформатированное расписание в Telegram

## Технологии

- **Python** — основной язык
- **aiogram** / **python-telegram-bot** — работа с Telegram API
- **imaplib** / **email** — проверка почты
- **JSON** — хранение расписаний автобусов и уроков

1. Установи зависимости:
   ```bash

2. Создай .env файл:

  EMAIL_PASSWORD=your_password
  IMAP_SERVER=imap.example.com

  SENDER_EMAIL=email_sender

3. Заполни JSON-файлы расписаний автобусов и уроков под свой колледж
4. Запусти бота:
